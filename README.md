# DESARROLLO-IA-LLM-


Desarrollo de una aplicación interactiva en Python con Streamlit que integra reconocimiento de voz, procesamiento de lenguaje natural y síntesis de voz para un chatbot conversacional en tiempo real. El sistema emplea Vosk para transcripción de audio, un modelo de lenguaje (Llama 3) mediante LangChain/Ollama para la generación de respuestas, y pyttsx3 para la síntesis de voz.
